Seq-gen

Seq-gen is a program that simulates sequence evolution by using current substitution models.


It was initially conceived to be used in the line of command. The objective of this website is to democratize Seq-gen by simplifying its usage


For more information:
http://tree.bio.ed.ac.uk/software/seqgen
https://snoweye.github.io/phyclust/document/Seq-Gen.v.1.3.2/Seq-Gen.Manual.html


Original article:
Rambaut A, Grassly NC, Comput Appl Biosci, 1997
